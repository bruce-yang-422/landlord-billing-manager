# 🏠 房東帳務管理工具

專為台灣包租代管房東打造的輕量化帳務管理工具。支援**共用台電電號的多戶累進電費分攤**，自動按比例計算各戶應付電費，水費依居住人數分攤，一鍵產出 LINE 報表。

## 🌟 核心功能

### ⚡ 台電累進電費分攤

- 共用電號（5F + 6F）輸入一次台電帳單，自動計算兩戶各自應付電費
- 採用**累進費率理論電費 + 差額按用電比例分攤**的公正計算方式
- 5F 用電量 = 台電總度數 − 6F 電表讀數（不需要 5F 獨立電表）
- 支援夏月／非夏月費率切換

### 💧 水費按人數分攤

- 輸入水費總額，自動依各戶居住人數比例分攤
- 人數設定長期記憶，不需每月重填

### 🏠 多戶獨立帳單

- 5F、6F 各自獨立存檔，歷史記錄可按樓層篩選
- 各戶獨立設定：租金、銀行收款資訊、租客備註、房東備註
- 各戶獨立的瓦斯費、管理費、其他費用輸入

### 📋 LINE 報表

- 一鍵產出含電費分攤說明的透明化 LINE 報表
- 租客備註顯示在報表中，房東備註僅供自己記錄

### 💾 資料管理

- 所有資料儲存在瀏覽器 localStorage，不上傳任何伺服器
- 支援 JSON 匯出備份、匯入還原
- 輸入值自動記憶，下次開啟不需重填

## 📖 使用方式

### 首次設定（僅需一次）

1. 點擊右上角「⚙️ 房客設定」
2. 分別填入 5F、6F 的租金、居住人數、銀行收款資訊
3. 在「備註」Tab 填入租客備註（會顯示在 LINE 報表）和房東備註（僅供記錄）
4. 點「💾 儲存房客設定」，面板自動收起

### 每月記帳

1. 確認帳單日期、選擇夏月／非夏月
2. 填入台電帳單金額、總度數
3. 填入 6F 本期電表讀數（上期讀數會自動帶入）
4. 電費分攤金額即時預覽
5. 填入本期水費總額，水費分攤金額即時預覽
6. 各戶分別填入瓦斯費、管理費、其他費用（如有）
7. 點「💾 存檔 5F 帳單」／「💾 存檔 6F 帳單」各自存檔
8. LINE 報表自動產出，點複製後貼給租客

## ⚡ 電費分攤計算說明

採用與台電計算機相同的**累進理論費用 + 差額分攤**方式：

```text
1. 分別計算 5F、6F 若獨立計費的理論電費（C5理、C6理）
2. 累進差額 ΔC = 台電帳單 − (C5理 + C6理)
3. 各戶分攤差額 = ΔC × (本戶度數 / 總度數)
4. 各戶應付 = 理論電費 + 分攤差額
5. 確保 5F + 6F = 台電帳單（四捨五入誤差由 6F 吸收）
```

此方式比「按平均單價」更公正，反映累進費率的實際成本分配。

## ⚖️ 113年7月新制法規

113年7月15日起，新簽租約電費計收須遵循實報實銷原則：

- 按度計費（有獨立電表）：不得超過台電帳單**當期每度平均電價**
- 違規超收最高可處 **50 萬元**罰鍰

本工具的累進分攤方式確保收費金額加總等於台電帳單，天然符合新制規定。
